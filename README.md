# TransitSense 🚍

## Project Overview
TransitSense is an intelligent bus tracking and crowd prediction system designed to help users choose less crowded and more efficient travel options.

## Features
- Real-time bus tracking (simulated)
- Crowd prediction (Low, Medium, High)
- Smart bus recommendation
- Admin dashboard (future scope)

## Tech Stack
- Backend: Python (FastAPI)
- Frontend: React
- Database: MongoDB (planned)

## How to Run
1. Go to backend folder
2. Run: uvicorn main:app --reload
3. Open: http://127.0.0.1:8000/docs
